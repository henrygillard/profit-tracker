  <implementation>
    lib/shopifyClient.js:
    - API_VERSION = '2025-10' (constant at top)
    - async shopifyGraphQL(shop, accessToken, query, variables = {})
    - POST to https://{shop}/admin/api/2025-10/graphql.json
    - Headers: Content-Type: application/json, X-Shopify-Access-Token: accessToken
    - If res.ok is false: throw Error(`Shopify GraphQL ${res.status}: ${text}`)
    - If json.errors: throw Error(JSON.stringify(json.errors))
    - Return json.data
    - module.exports = { shopifyGraphQL }

    lib/profitEngine.js:
    - THIRD_PARTY_FEE_RATES constant map (Basic, Grow, Advanced, Plus)
    - getThirdPartyFeeRate(planDisplayName): lookup with default 0.02
    - getCOGSAtTime(prisma, shop, variantId, processedAt): async, query productCost, return parseFloat(cost.costAmount) or null
    - calculateOrderProfit(order, options): pure function
      - order shape: { currentTotalPrice, totalRefunded, lineItems, paymentGateway, shippingCost, shopifyPaymentsFee }
      - lineItems: [{ variantId, quantity, unitPrice, cogs }] (cogs already resolved, null if unknown)
      - revenueNet = parseFloat(currentTotalPrice) (already the post-refund net from Shopify's currentTotalPriceSet)
      - cogsKnown = lineItems.every(li => li.cogs !== null)
      - cogsTotal = cogsKnown ? sum(li.quantity * li.cogs) : null
      - For refund proportional adjustment: multiply cogsTotal by (revenueNet / totalPrice) when totalRefunded > 0
      - feesTotal = shopifyPaymentsFee if paymentGateway includes 'shopify_payments', else getThirdPartyFeeRate(planName) * revenueNet
      - netProfit = cogsKnown ? revenueNet - cogsTotal - feesTotal - shippingCost : null
      - Returns object matching OrderProfit fields (revenueNet, cogsTotal, feesTotal, shippingCost, netProfit, cogsKnown)
    - module.exports = { shopifyGraphQL is NOT here — separate files }
    - module.exports = { calculateOrderProfit, getCOGSAtTime, getThirdPartyFeeRate, THIRD_PARTY_FEE_RATES }

    CRITICAL anti-patterns to avoid:
    - Never coerce null COGS to 0 — use null throughout
    - calculateOrderProfit is a pure function — no Prisma calls, no I/O (getCOGSAtTime is separate and called by sync code before passing cogs-resolved lineItems in)
  </implementation>
