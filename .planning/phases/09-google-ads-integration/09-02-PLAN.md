From 09-RESEARCH.md — callback flow (exact sequence):
```
1. Verify OAuthState CSRF
2. client.getToken(code) → { tokens: { refresh_token, access_token } }
3. fetch listAccessibleCustomers → resourceNames
4. If 1 ID: use it directly. If multiple: GAQL query customer.manager on each, pick first non-manager
5. encrypt(tokens.refresh_token) → store in AdConnection (platform='google')
6. Delete OAuthState, redirect to /admin?shop=x
```

Create `routes/google-ads-auth.js` using the patterns from `routes/ads-auth.js` as structural template. Key differences from Meta version:

**GET /google-ads/auth:**
- Iframe escape block: copy verbatim from ads-auth.js lines 27-51 with path changed to `/google-ads/auth`
- Check `GOOGLE_ADS_CLIENT_ID` env var (log error + 500 if missing)
- Create OAuthState CSRF (same prisma.oAuthState.create + deleteMany cleanup pattern)
- Build auth URL using `OAuth2Client.generateAuthUrl()` with `access_type: 'offline'`, `prompt: 'consent'`, `scope: ['https://www.googleapis.com/auth/adwords']`
- `res.redirect(authUrl)`

**GET /google-ads/callback:**
1. Verify `code` and `state` params (400 if missing)
2. `prisma.oAuthState.findUnique({ where: { state } })` — 400 if not found
3. `await prisma.oAuthState.delete({ where: { state } })`
4. `const client = new OAuth2Client(GOOGLE_ADS_CLIENT_ID, GOOGLE_ADS_CLIENT_SECRET, SHOPIFY_APP_URL + '/google-ads/callback')`
5. `const { tokens } = await client.getToken(code)` — 500 if throws
6. Call `listAccessibleCustomers`:
   ```javascript
   const listRes = await fetch('https://googleads.googleapis.com/v18/customers:listAccessibleCustomers', {
     headers: {
       Authorization: `Bearer ${tokens.access_token}`,
       'developer-token': process.env.GOOGLE_ADS_DEVELOPER_TOKEN,
     },
   });
   const { resourceNames } = await listRes.json();
   const customerIds = (resourceNames || []).map(r => r.replace('customers/', ''));
   ```
7. Account selection: if `customerIds.length === 1`, use `customerIds[0]` directly. If multiple, iterate and GAQL query `SELECT customer.id, customer.manager FROM customer LIMIT 1` for each — pick first where `customer.manager === false`. Fallback to `customerIds[0]` if all are managers.
8. `const encryptedToken = encrypt(tokens.refresh_token)`
9. `await prisma.adConnection.upsert({ where: { shop_platform: { shop, platform: 'google' } }, update: { encryptedToken, accountId: selectedId }, create: { shop, platform: 'google', encryptedToken, accountId: selectedId } })`
10. `res.redirect('/admin?shop=' + encodeURIComponent(shop))`

Error handling: wrap entire callback in try/catch. Log + 500 on unexpected errors.

Do NOT add a POST /connect or DELETE /disconnect route — those belong to routes/ads-auth.js (connect) and routes/ads.js (disconnect) respectively.
